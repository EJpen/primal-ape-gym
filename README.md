# Primal Ape Gym

> Unleash Your Inner Ape. Built Different. Train Like a Silverback.

## Purpose
A high-performance fitness center website designed to showcase elite training programs, membership plans, and gym facilities. The site brings the gym's "Strength is Instinct" philosophy to life through an immersive, primal jungle-themed digital experience.

## Tech Stack
- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **GSAP (GreenSock)** - Industry-standard animation library for scroll and hover effects
- **Lenis** - Smooth scrolling library for seamless navigation
- **Google Fonts** - Bebas Neue, Inter, and Oswald typography
